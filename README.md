# Diabetes ML Analysis

This project focuses on analyzing a diabetes dataset and training machine learning models to predict diabetes risk.

## Overview
The objective of this project is to build and evaluate machine learning models using healthcare data. The workflow includes data preprocessing, feature selection, model training, and performance evaluation.

## Dataset
The dataset includes multiple health-related features such as:
- Age
- Gender
- Glucose level
- BMI
- Blood pressure
- Family medical history

## Workflow
- Load and explore the dataset
- Clean and preprocess the data
- Encode categorical variables
- Select relevant features
- Split data into training and testing sets
- Train machine learning models
- Evaluate model performance

## Models
- Random Forest
- Support Vector Machine (SVM)
- XGBoost

## Technologies Used
- Python
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
- xgboost

## Project Structure
- diabetes_model_training.ipynb
